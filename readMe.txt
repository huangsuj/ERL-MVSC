Title: Embedding Regularizer Learning for Multi-View Semi-Supervised Classification（ERL-MVSC）
--------------------------------------------------------------

The project deals with multi-view semi-supervised classification problem. The work has been published in IEEE Transactions Image Processing. 
If you have any questions, please do not hesitate to contact us. (Email: sxxhap@163.com)

To run the ERL-MVSC algorithm you can simply execute the script: fsClassification_semiERLMSC.m.

The details of the main folders are given below,
- semiERLMVSC : Conatins implementation of ERL-MVSC.
- fsClassification_semiERLMSC: get the classification performance.
